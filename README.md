# data-structures-algorithms
Repository for data structures and algorithms implemented in cpp

## Usage 
Make sure you have [C++](https://isocpp.org/) installed on your machines.  

## Compilation

```bash
g++ file_name
```
## Execution

For Linux Machines  
```bash
./a.out
```

For Windows Machines
```bash
a.exe
```
